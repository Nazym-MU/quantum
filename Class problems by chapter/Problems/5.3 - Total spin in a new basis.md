Express the total-spin $s = 0$ state of two spin-$\frac{1}{2}$ particles given in (5.31) in terms of the states $\ket{+\mathbf{n}, -\mathbf{n}}$ and $\ket{-\mathbf{n}, +\mathbf{n}}$, where for a single spin-$\frac{1}{2}$ particle:
$$\ket{+\mathbf{n}} = \cos\frac{\theta}{2}\ket{+z} + e^{i\phi}\sin\frac{\theta}{2}\ket{-z}$$
$$\ket{-\mathbf{n}} = \sin\frac{\theta}{2}\ket{+z} - e^{i\phi}\cos\frac{\theta}{2}\ket{-z}$$
---
## Solution
### Represent $\ket{+z}$ in $\mathbf{\hat{n}}$ basis:
$$\braket{+\mathbf{n}|+z}\ket{+\mathbf{n}} = \left(\cos\frac{\theta}{2}\braket{+z} + e^{-i\phi}\sin\frac{\theta}{2}\braket{-z}\right)\ket{+z}\ket{+\mathbf{n}} = \cos\frac{\theta}{2}\ket{+\mathbf{n}}$$
$$\braket{-\mathbf{n}|+z}\ket{-\mathbf{n}} = \left(\sin\frac{\theta}{2}\braket{+z} - e^{-i\phi}\cos\frac{\theta}{2}\braket{-z}\right)\ket{+z}\ket{-\mathbf{n}} = \sin\frac{\theta}{2}\ket{-\mathbf{n}}$$
$$\boxed{\ket{+z} = \cos\frac{\theta}{2}\ket{+\mathbf{n}} + \sin\frac{\theta}{2}\ket{-\mathbf{n}}}$$
### Represent $\ket{-z}$ in $\mathbf{\hat{n}}$ basis:
$$\braket{+\mathbf{n}|-z}\ket{+\mathbf{n}} = e^{-i\phi}\sin\frac{\theta}{2}\ket{+\mathbf{n}}$$
$$\braket{-\mathbf{n}|-z}\ket{-\mathbf{n}} = -e^{-i\phi}\cos\frac{\theta}{2}\ket{-\mathbf{n}}$$
$$\boxed{\ket{-z} = e^{-i\phi}\left(\sin\frac{\theta}{2}\ket{+\mathbf{n}} - \cos\frac{\theta}{2}\ket{-\mathbf{n}}\right)}$$
### Express the singlet state $\ket{0,0}$:
The singlet state is:

$$\ket{0,0} = \frac{1}{\sqrt{2}}\ket{+z,-z} - \frac{1}{\sqrt{2}}\ket{-z,+z}= \frac{1}{\sqrt{2}}\left(\ket{+z}_1 \otimes \ket{-z}_2\right) - \frac{1}{\sqrt{2}}\left(\ket{-z}_1 \otimes \ket{+z}_2\right)$$
Substituting expressions:

$$= \frac{1}{\sqrt{2}}\left(\cos\frac{\theta}{2}\ket{+\mathbf{n}}_1 + \sin\frac{\theta}{2}\ket{-\mathbf{n}}_1\right) \otimes e^{-i\phi}\left(\sin\frac{\theta}{2}\ket{+\mathbf{n}}_2 - \cos\frac{\theta}{2}\ket{-\mathbf{n}}_2\right)$$

$$- \frac{1}{\sqrt{2}}\left(e^{-i\phi}\left(\sin\frac{\theta}{2}\ket{+\mathbf{n}}_1 - \cos\frac{\theta}{2}\ket{-\mathbf{n}}_1\right)\right) \otimes \left(\cos\frac{\theta}{2}\ket{+\mathbf{n}}_2 + \sin\frac{\theta}{2}\ket{-\mathbf{n}}_2\right)$$
$$= \frac{1}{\sqrt{2}}\left(e^{-i\phi}\sin\frac{\theta}{2}\cos\frac{\theta}{2}\ket{+\mathbf{n},+\mathbf{n}} - e^{-i\phi}\cos^2\frac{\theta}{2}\ket{+\mathbf{n},-\mathbf{n}} + e^{-i\phi}\sin^2\frac{\theta}{2}\ket{-\mathbf{n},+\mathbf{n}} - e^{-i\phi}\sin\frac{\theta}{2}\cos\frac{\theta}{2}\ket{-\mathbf{n},-\mathbf{n}}\right)$$
$$- \frac{1}{\sqrt{2}}\left(e^{-i\phi}\sin\frac{\theta}{2}\cos\frac{\theta}{2}\ket{+\mathbf{n},+\mathbf{n}} + e^{-i\phi}\sin^2\frac{\theta}{2}\ket{+\mathbf{n},-\mathbf{n}} - e^{-i\phi}\cos^2\frac{\theta}{2}\ket{-\mathbf{n},+\mathbf{n}} - e^{-i\phi}\sin\frac{\theta}{2}\cos\frac{\theta}{2}\ket{-\mathbf{n},-\mathbf{n}}\right)$$
The $\ket{+\mathbf{n},+\mathbf{n}}$ and $\ket{-\mathbf{n},-\mathbf{n}}$ terms cancel:
$$\ket{0,0} = \frac{e^{-i\phi}}{\sqrt{2}}\left(-\cos^2\frac{\theta}{2} - \sin^2\frac{\theta}{2}\right)\ket{+\mathbf{n},-\mathbf{n}} + \frac{e^{-i\phi}}{\sqrt{2}}\left(\sin^2\frac{\theta}{2} + \cos^2\frac{\theta}{2}\right)\ket{-\mathbf{n},+\mathbf{n}}$$
Using $\sin^2\frac{\theta}{2} + \cos^2\frac{\theta}{2} = 1$:
$$\boxed{\ket{0,0} = \frac{e^{-i\phi}}{\sqrt{2}}\left(\ket{-\mathbf{n},+\mathbf{n}} - \ket{+\mathbf{n},-\mathbf{n}}\right)}$$

The singlet state is antisymmetric under particle exchange in **any** spin basis, not just the $z$-basis
