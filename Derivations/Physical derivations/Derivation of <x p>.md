## ⟨x|p⟩
Starting from eigenvalue equation p̂ₓ|p⟩ = p|p⟩, project onto ⟨x|:
$$\langle x|\hat{p}_x|p\rangle = p\langle x|p\rangle$$

Using p̂ₓ = -iℏ(∂/∂x) in position basis:
$$-i\hbar\frac{\partial}{\partial x}\langle x|p\rangle = p\langle x|p\rangle$$

This differential equation has solution:
$$\langle x|p\rangle = Ne^{ipx/\hbar}$$

Normalization condition ⟨p'|p⟩ = δ(p'-p) determines N = 1/√(2πℏ).
