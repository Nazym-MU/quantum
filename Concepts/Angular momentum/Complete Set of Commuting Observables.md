A **complete set of commuting observables (CSCO)** is a set of operators {Â₁, Â₂, ..., Âₙ} such that:
1. All operators commute with each other: $[\hat A_i, \hat A_j] = 0$
2. The eigenvalues uniquely specify each state
3. No proper subset has this property (minimality)
## For Central Potential
Hamiltonian: $\hat H = \hat p^2/2μ + V(r)$
**CSCO**: $\{\hat H, \hat L^2, \hat L_z\}$
States labeled: |E, l, m⟩ where
- $Ĥ|E, l, m⟩ = E|E, l, m⟩$
- $\hat L^2|E, l, m⟩ = l(l+1)ℏ^2|E, l, m⟩$
- $\hat L_z|E, l, m⟩ = mℏ|E, l, m⟩$
## Degeneracy and CSCO
For each energy E and angular momentum $l$, there are $2l+1$ states with different $m$ values $(m = -l, -l+1, ..., l-1, l)$. The quantum number $m$ is needed to break this degeneracy and uniquely label states.
## With Spin
For real hydrogen atom, must include electron and proton spin. Example CSCO including hyperfine interaction: $\{\hat H, \hat L^2, \hat S^2, \hat L_z, \hat S_z\}$ where $\hat S = \hat S_1 + \hat S_2$.
